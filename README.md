## Project Summary
Smart Campus Assistant is a multi-page prototype designed to help students quickly 
access essential university information such as:
- Login screen (demo mode)
- Timetable (with day filter)
- News feed + news details
- Events list + event details
- Contacts directory
- Campus map page
- Quick links

The prototype is built using **HTML, CSS, and JavaScript** and does not include 
real authentication or backend logic. All interactions are client-side only.

## How to run the prototype
1. Download the repository / clone it  
2. Open **index.html** in any browser  
3. Use the top navigation bar to move between screens  

The prototype simulates:
- login behaviour  
- timetable filtering  
- opening news and events  
- navigating between sections
